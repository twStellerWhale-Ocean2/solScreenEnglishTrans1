# Changelog

版本依語意化版號（SemVer）。版號於 PR merge 當下釘選。

## [0.30.0] - 2026-07-06

Issue #97 增量：發音練習卡片改麥克風鈕與成績框（含 `gpt-audio-mini` 模型修正）＋UI 微調。

### 變更
- **發音練習回饋重設計（spec#10）**：「我的筆記」卡片行尾由「播音鈕＋燈泡」改為 **播音鈕 ▶｜麥克風鈕｜成績框**——
  麥克風鈕**按住錄音／放開送 AI 評分**（錄音中轉紅）；**成績框五態**＝未練灰「—」／**最佳分**
  （<門檻**紅色空框**、≥門檻**綠底＋✓**，非僅以顏色分辨）／錄音中**藍色音量條依即時麥克風音量由下而上**／
  評分中 **spinner 轉圈**／得分先閃「這次分數」再回落最佳分。`PracticeScore` 語意由「最後分」改為「**最佳分**」
  （取 Max；相容舊檔）；曾達標即恆綠，直到 Clear Practice 歸零。
- **播音鈕改圓鈕**：與麥克風鈕同款圓形圖示鈕，明示可按、外觀一致。

### 修正
- **發音評分模型改 `gpt-audio-mini`**：OpenAI 音訊輸入模型（`gpt-audio` 系列）**不支援 structured outputs**，
  改為去除 `response_format`、以提示要求 JSON 並穩健解析（容錯 ```json 圍欄、取第一個 `{…}` 物件、分數夾限 0–100），
  修正舊模型名 `gpt-4o-mini-audio-preview` 404 而無法評分之問題。

### 介面微調
- **About 頁改橫向**：女孩圖在左、資訊在右，Change Log 於其下整條寬（不再上下鬆散或左右擠窄）。
- **主視窗與查詢結果視窗**加**右下角縮放握把**（`ResizeMode=CanResizeWithGrip`）。
- **查詢結果卡**中英播放鈕文字統一為「**Play**」（原 Play sentence／Play Chinese）。

## [0.29.0] - 2026-07-06

Issue #95 增量：我的筆記新增發音練習燈泡（清空全部改為清空練習紀錄）。

### 新增
- **筆記發音練習（spec#10）**：「我的筆記」卡片播音鈕旁新增**燈泡鈕**——按住錄音、放開送 AI 評分，
  唸出的分數達「選項」分頁設定的**及格門檻**（預設 80）即點亮燈泡並記錄分數（隨 `notes.json` 留存、重啟保留）；
  放開後以 toast 明示分數（如「Pronunciation 88 / 80 ✓」），無麥克風／OS 隱私未授權／錄音太短／評分失敗
  各自明確提示、燈泡不誤亮；評分中呈忙碌態、同時至多一個練習。
- **發音評分（[techItem發音評分]）**：`IPronunciationAssessor`／`PronunciationService`——以 OpenAI 音訊輸入模型
  （`paramPronModel`，預設 `gpt-4o-mini-audio-preview`）評分，沿用既有 `OPENAI_API_KEY`（環境變數、不落地）
  與查詢層之逾時／有限次重試／降級。錄音經 NAudio 擷取（16kHz 單聲道）、僅存記憶體、不落地。
- **選項頁「發音練習」區塊**：可調及格門檻（0–100，滑桿＋數值輸入）與評分模型（`paramPronPassThreshold`／
  `paramPronModel`，隨 appsettings 保存、相容舊檔；門檻改動即時依新值重判燈泡）。

### 變更
- **右欄頂「Clear All」改為「Clear Practice」**：由「清除全部筆記」改為「**清空該夾練習紀錄**」（所有燈泡歸零、
  不刪筆記）；批次刪除筆記改由整夾刪除（`Del`）承接、逐筆刪除走右鍵。`NoteEntry` 新增 `PracticeScore` 欄
  （-1＝未練，舊 `notes.json` 相容）。

## [0.28.0] - 2026-07-06

- **指定快捷鍵不再誤觸翻譯**：於「選項」分頁按「Change」進入監聽指定喚起鍵／點選鍵期間，暫停全域喚起熱鍵

Issue #86 增量：新增第二個熱鍵直接點選擷取（取代遮罩內雙擊）。

- **直接點選擷取熱鍵**（預設 `Alt+A`）：按下即**直接**截取整個虛擬桌面、於目前游標位置畫紅色標記，

- 新增組態 `paramHotkeyPoint`（預設 `Alt+A`；舊 appsettings 無此鍵時套預設，向後相容）。

## [0.21.0] - 2026-07-05

Issue #55 增量：查詢結果頁可設定筆記加入預設（資料夾、底色）＋智能配色。

### 新增
- **加入至指定資料夾**：查詢結果視窗「加入我的筆記」下方新增「加入至 [資料夾▾]」——可選「（使用中情境）」
  （以使用中情境名為資料夾、不存在則建立；無情境則預設夾「我的筆記」）或任一既有頂層資料夾；選擇記憶為預設。

  （例：戰士台詞粉紅、法師粉藍）；非空時附加於查詢提示，AI 回結果時順帶建議底色，於結果頁色塊**自動預選**
